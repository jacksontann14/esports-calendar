Setup

  1. iPhone: tap any "Subscribe" link above — iOS opens the Calendar app and prompts you to confirm.
  2. Google Calendar: left sidebar → Other calendars → + → From URL, then paste the feed's https:// link.
  3. Outlook: Add calendar → Subscribe from web, paste the same link.
  4. Feeds refresh once a day automatically — your calendar app re-checks them on its own schedule, so new matches can take a few hours to appear.
